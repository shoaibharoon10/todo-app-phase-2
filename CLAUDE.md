# Project Governance: Phase 2 (Full-stack Evolution)

## Core Rule
- **NO MANUAL CODING:** Every implementation must be driven by specifications in the `/specs` folder.
- If an error occurs, refine the spec instead of editing the code.

## Current Focus: Phase 2
- **Objective:** Evolve from a Python CLI to a Full-stack Web Application.
- **Backend:** FastAPI with SQLModel for database interactions.
- **Frontend:** Next.js (App Router) with Tailwind CSS.
- **Database:** Neon DB (Serverless PostgreSQL).
- **Authentication:** Better Auth.

## Environment & Tooling
- **Python:** Use Python 3.13 with `uv` for backend dependency management.
- **Node.js:** Use `npm` for frontend dependency management in the `/frontend` directory.
- **Urdu Support:** Maintain UTF-8 encoding for Urdu language support in the Web UI.
